# Worker registration
GUI application for CRUD operations on workers in a database

### Prerequisites

-JDK 1.8

-Maven

### Dependencies:
-H2 
-Hibernate

## How to run

Run `mvn install` in root directory

Run `java -jar target/jfx/app/Workers-1.0-SNAPSHOT-jfx.jar`
